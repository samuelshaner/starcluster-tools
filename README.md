# starcluster-tools
Fun tools built upon StarCluster.

This Repository has the following that contain useful tools for working with StarCluster:

- config -> Files that contain config file blocks to insert into one's config file to create specific cluster types, plugins, etc.
- utilities -> General utilities for performing actions like looking up the spot prices of all instance types.
- plugins -> Plugin files that can be used to initialize a cluster with proper modules/libraries for running a certain code or performing some task.
